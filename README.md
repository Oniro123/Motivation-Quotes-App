# **Motivational Quotes App**

A full-stack web application that allows users to share and discover motivational quotes. Built with Node.js, PostgreSQL, and vanilla CSS.

# **Features**

**User Authentication:** Secure registration and login system using Passport.js
**Quote Management:** Full CRUD functionality for creating, reading, updating, and deleting personal quotes
**Community Quotes:** You can browse motivational quotes shared by other users
**Daily Quote:** Automatically fetches and displays an inspirational quote every hour using the ZenQuotes.io API
**Secure Data Storage:** User passwords are hashed with bcrypt before storage in PostgreSQL

# **Tech Stack**
**Backend:**

Node.js
Express.js
PostgreSQL
Passport.js (authentication)
bcrypt (password hashing)

**Frontend:**

HTML/CSS
JavaScript

# **External API:**

ZenQuotes.io - Daily inspirational quotes
